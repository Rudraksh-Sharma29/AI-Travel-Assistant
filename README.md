# AI Travel Assistant ✈️

An AI-powered Travel Assistant built using Python and Google ADK that helps users with travel-related queries, recommendations, and conversational assistance.

## Features

- Conversational AI travel assistant
- Smart travel recommendations
- Interactive chat responses
- Modular Python structure
- Easy to extend with APIs and additional tools

## Tech Stack

- Python
- Google ADK
- Gemini API
- FastAPI (planned/optional)
- Git & GitHub
